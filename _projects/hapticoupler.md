The HaptiCoupler is an open-source system that enables bidirectional tactile interaction with audio-driven synthesisers.

The PCB designs, firmware, and documentation is available [on GitHub](https://github.com/davisonaudio/HapticInstrumentWorkshop).

The system enables simultaneous haptic feedback and tactile sensing with a single voicecoil transducer (acting as both a sensor and actuator simultaneously)

It is primarily designed as an interface for audio-driven resonant synthesis models.
